Portfolio Saya

Website Toko Besi

Website TB. Rama Jaya adalah landing page statis yang dibangun menggunakan HTML, CSS, dan JavaScript yang menampilkan profil toko besi dan bahan bangunan dengan desain responsif, sederhana, dan profesional. Berisi informasi produk, keunggulan, tentang kami, testimoni pelanggan, serta kontak agar memudahkan pengunjung mengenal dan terhubung dengan toko.

Brain Application

Brain Application adalah aplikasi manajemen tugas dan jadwal yang dibangun menggunakan JavaFX dengan menerapkan konsep CRUD (Create, Read, Update, Delete). Aplikasi ini memungkinkan pengguna menambahkan tugas baru, menampilkan daftar tugas, memperbarui detail tugas, dan menghapus tugas sesuai kebutuhan. Dengan antarmuka grafis sederhana namun interaktif, Brain Application membantu pengguna mengelola aktivitas harian secara lebih terstruktur dan efisien.

Game Tebak Angka

Tebak Angka adalah aplikasi game interaktif berbasis JavaFX di mana pengguna diminta menebak angka acak antara 1–100. Aplikasi ini menampilkan input field, tombol aksi, serta jumlah percobaan yang dilakukan. Dengan memanfaatkan antarmuka grafis JavaFX, game ini memberikan pengalaman sederhana namun menyenangkan dalam melatih logika dan konsentrasi pengguna.

Website Bengkel Las

Website ini dirancang dengan tampilan modern, responsif, dan mudah diakses di berbagai perangkat, sehingga memudahkan pelanggan maupun mitra dalam mendapatkan informasi secara cepat dan jelas.
